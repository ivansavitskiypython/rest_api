To import xml to "/my_posts/import_xml" use this structure:

<?xml version="1.0" encoding="UTF-8"?>
<articles>
  <list-item>
    <body>Post body from xml</body>
    <title>Post title from xml</title>
  </list-item>
  <list-item>
    <body>Пост для тестирования 2. Импортирован из ХМЛ</body>
    <title>Пост из ХМЛ 2</title>
  </list-item>
</articles>

A timestamp is automatically inserted when you create an article.

Also I renamed the model "post" in "article".
